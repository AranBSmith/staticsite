This project investigated how effective the Intelligent Trial and Error (ITE) algorithm would be in allowing the Rhex robot to adapt to platform damage, and novel environments. In the context of robust robot swarms, the Rhex platform could prove to be very useful with an effective adaptive algorithm.

The ITE algorithm utilises the MAP-Elites evolutionary algorithm to generate a repertoire of Rhex gaits. It then uses Bayesian optimization to find the best gait for its current setting.

In this project I designed and implemented Rhex control schemes which the MAP-Elites algorithm would evolve, and the ITE algorithm would utilise. One of the challenges with this project was that I needed to run programs on machines with at least 8 cores, for several days. I achieved this with Google Cloud Platform (GCP) to generate several maps. After running the evolutionary algorithms for days, and hundreds of experiments I showed that this approach can be used to allow Rhex to adapt in novel environments and to different damage cases.
